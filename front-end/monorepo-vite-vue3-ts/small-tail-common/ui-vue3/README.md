# 小尾巴前端通用 Vue3 UI 组件库

## 安装

### npm

```bash
npm install st-common-ui-vue3
```

### yarn

```bash
yarn add st-common-ui-vue3
```

### pnpm

```bash
pnpm install st-common-ui-vue3
```
