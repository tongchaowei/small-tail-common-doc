# 小尾巴前端通用核心库

在该库中，对 JavaScript 基础类型、函数、数组等进行了封装，并提供了一些常用的工具函数。

## 安装

### npm

```bash
npm install st-common-core
```

### yarn

```bash
yarn add st-common-core
```

### pnpm

```bash
pnpm install st-common-core
```
